Glossary
========

.. glossary::

  VDOM
    A :ref:`media type definition <VDOM Mimetype>` for Virtual Document Object Models.

  Pure Element
    A VDOM element which is defined using a pure function.
    Read more in :ref:`Pure Elements`.

  Stateful Element
    A VDOM element which is defined using a function, but which uses
    :meth:`@element <idom.core.element.element>` to manage some state.
    Read more in :ref:`Stateful Elements`

  Class Element
    A VDOM element chich was defined by subclassing
    :class:`AbstractElement <idom.core.element.AbstractElement>`.
    Read more in :ref:`Class Elements`.
